## Список данных и способы защиты

| Категория данных                          | Конкретные поля                               | Способ защиты                                |
|:------------------------------------------|:----------------------------------------------|:---------------------------------------------|
| **PII (персональные)**                    | ФИО, телефон, email, адрес                    | Псевдонимизация                              |
| **PII (чувствительные)**                  | Дата рождения, место работы                   | Шифрование + RBAC/ABAC                       |
| **Медицинская тайна**                     | Диагнозы, хронические заболевания, анализы    | Шифрование + обезличивание                   |
| **Платёжные данные**                      | Сумма, способ оплаты, номер карты (если есть) | Маскирование                                 |
| **Скан-копии**                            | Договоры, согласия, паспорт                   | Шифрование файлов                            |
| **Согласия на обработку ПДн**             | Статус согласия, дата отзыва, версия формы    | Шифрование + retention + удаление по запросу |
| **Лабораторные данные**                   | Результаты анализов, реестры                  | Обезличивание + шифрование                   |
| **Логи доступа**                          | Кто, когда, к каким данным обращался          | Подпись/контроль целостности + WORM-хранение |
| **Идентификаторы сессий и техметаданные** | IP, user-agent, device id, session id         | Маскирование + ограниченный срок хранения    |
